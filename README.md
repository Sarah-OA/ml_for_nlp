# ml_for_nlp
This repository contains my final project for the Machine Learning for Natural Language Processing course at ENSAE Paris.
For this project, I studied the Nationalization vs. Privatization debate in the legislative elections. I used topic modelling and sentiment analysis to better understand the debate and the positions of each party or political block, as well as their evolution.

## Organization of the repository
data: contains the raw texts and metadata for the legislative elections of 1981, 1988 and 1993, as well as processed data for the project. 
figures: various figures generated from the notebook, available in the report.
Code - Nationalization - Privatization Sentiment Analysis.ipynb: final notebook of the project.
Nationalization - Privatization Sentiment Analysis - report.pdf: report of the results.
stop_word_fr.txt: list of stop words used for topic modelling.
